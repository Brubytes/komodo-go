# Code of Conduct

## Our Commitment

We want this project to be an open, welcoming, and respectful community for everyone.

## Expected Behavior

- Be respectful and constructive in discussions and reviews.
- Assume good intent and give actionable feedback.
- Focus on technical topics and project goals.
- Help keep discussions inclusive and beginner-friendly.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or derogatory comments.
- Publishing private information without consent.
- Repeatedly derailing discussions or spamming.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject comments, commits, issues, pull requests, and other
contributions that violate this policy.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through a
private channel when possible. If no private channel is available, open an issue with
minimal details and request a secure follow-up.
